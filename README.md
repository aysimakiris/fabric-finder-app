# fabric-finder-app
Fabric Finder is a concept app designed to help users discover, compare, and select fabrics across different brands. The platform aims to simplify decision-making by making material properties, pricing, and sustainability attributes more transparent and accessible.
Problem
Finding the right fabric is often fragmented and time-consuming. Users need to browse multiple brand websites, compare inconsistent information, and rely on limited filtering options. This creates friction in both creative and purchasing processes.
Solution
The app brings together fabrics from multiple sources into a single interface, allowing users to:
Search fabrics by type, texture, and use case
Compare materials across brands
Access standardized information (composition, durability, sustainability)
Make more informed and efficient decisions
My Role
Product concept development
UX research & insight synthesis
User journey mapping
Interface design
Process
The project was developed through:
Identifying gaps in current fabric discovery experiences
Exploring user needs in creative and retail contexts
Designing a comparison-based interface
Iterating on usability and clarity
Outcome
The result is a user-centered concept that improves transparency and accessibility in fabric selection, supporting both professionals and everyday users.
